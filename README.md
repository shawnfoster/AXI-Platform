# AXI Platform

AXI Platform is a governed runtime and reconstruction repository.

Current runtime status:

- Runtime Foundation is implemented through `M18 Runtime API`.
- Implemented runtime components include the Platform Object Model,
  Registry Foundation, Capability Registry, Service Registry, Event Bus,
  Dependency Resolver, Validation Framework, Plugin Loader, Application
  Registry, Engine Registry, Pipeline Runtime, Runtime CLI, and
  Runtime API.
- `M17 Runtime CLI` remains implemented and validated as the local
  command execution surface.
- `M18 Runtime API` is implemented, validated, and extends the current
  governed runtime through the local programmatic execution surface.
- No runtime implementation is claimed beyond `M18`.

Authoritative readiness is tracked in:

- `Governance/DependencyMatrix.md`
- `Governance/RuntimeRoadmap.md`
